# Tenzies 🎲
Click the dices that you want to hold, the roll and select as many identical dices as you have untill
you don't get all the dices with the same number

# Technologies 
* React
* Tailwind
* Vite

# How to run the project 
* Clone Repo
* Install dependencies: npm install
* Run the development server: npm run dev
* Open in your browser: http://localhost:5173
